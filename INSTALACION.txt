  Guía Rápida de Instalación

REQUISITOS PREVIOS:
-------------------
✓ XAMPP instalado (Apache + MySQL + PHP)
✓ Navegador web moderno (Chrome, Firefox, Edge)

PASOS DE INSTALACIÓN:
--------------------

1. INICIAR SERVICIOS
   - Abrir el Panel de Control de XAMPP
   - Iniciar Apache (botón "Start")
   - Iniciar MySQL (botón "Start")

2. CREAR LA BASE DE DATOS
   - Abrir phpMyAdmin: http://localhost/phpmyadmin
   - Hacer clic en "Nueva" base de datos
   - Nombre: dashboard_admin
   - Cotejamiento: utf8mb4_unicode_ci
   - Hacer clic en "Crear"

3. IMPORTAR LAS TABLAS
   - Seleccionar la base de datos "dashboard_admin"
   - Hacer clic en la pestaña "Importar"
   - Hacer clic en "Seleccionar archivo"
   - Buscar y seleccionar el archivo: database.sql
   - Hacer clic en "Continuar"
   - Esperar mensaje de éxito

4. ACCEDER AL SISTEMA
   - Abrir el navegador
   - Ir a: http://localhost/dashboard-admin/
   - Aparecerá la pantalla de login

5. INICIAR SESIÓN
   Usar cualquiera de estas credenciales:

   ADMINISTRADOR (acceso completo):
   Email: admin@dashboard.com
   Password: admin123

   MODERADOR (crear/editar usuarios):
   Email: moderador@dashboard.com
   Password: admin123

   USUARIO (solo ver y editar perfil):
   Email: usuario@dashboard.com
   Password: admin123

SOLUCIÓN DE PROBLEMAS:
----------------------

❌ Error: "Could not connect to MySQL"
   → Verificar que MySQL esté corriendo en XAMPP
   → Verificar puerto (por defecto 3306)

❌ Error: "Database not found"
   → Crear la base de datos "dashboard_admin" en phpMyAdmin
   → Importar el archivo database.sql

❌ La página no carga correctamente
   → Verificar que Apache esté corriendo
   → Verificar la URL: http://localhost/dashboard-admin/
   → Limpiar caché del navegador (Ctrl + F5)

❌ No se ven los estilos CSS
   → Verificar que exista la carpeta: assets/css/
   → Verificar que exista el archivo: assets/css/style.css
   → Limpiar caché del navegador

❌ Error: "Access denied for user 'root'"
   → Editar config.php
   → Ajustar credenciales de MySQL si son diferentes
